Quick Start Guide

Getting Started

  • Launch the app — it opens in landscape on your iPad
  • Tap Add Student in the left sidebar to create your first student
  • Set each student's level (SL or HL) on the Roster tab
  • Navigate between tabs using the icons at the top of the screen

Saving Your Data

  • Data saves automatically to your device after every change
  • Use Export in the sidebar footer to download a JSON backup
  • Use Push to iCloud to sync across devices

Using on Desktop

  • Open dashboard.html in Chrome or Safari on your computer
  • Use Open Folder to save to your iCloud Drive folder
  • Both iPad and desktop will read from the same sync file

Frequently Asked Questions

Where is my data stored?

All data is stored locally on your iPad in the app's browser storage. Nothing is sent to any external server. When you use iCloud sync, data is stored in your personal iCloud Drive account.

How do I sync between iPad and my computer?

On iPad, tap Push to iCloud in the sidebar footer. On your Mac, open the dashboard HTML file in Chrome, click Open Folder, and select the iCloud Drive folder for this app. Both devices will read and write to the same JSON file.

Can I use this with multiple classes?

Yes. Use the class selector at the top of the sidebar to switch between classes. Each class has its own roster, assessments, and data. You can add as many classes as you need.

How do I back up my data?

Tap Export in the sidebar footer. This downloads a JSON file containing all your classes, students, scores, and settings. To restore, use Import and select the JSON file. We recommend backing up regularly.

What happens if I delete the app?

Deleting the app will remove all locally stored data. Make sure to export a backup or push to iCloud before uninstalling. If you have an iCloud backup, you can restore by reinstalling and pulling from iCloud.

Can students see my data?

No. This app is for teacher use only. There is no student-facing view, no shared access, and no online component. Data only exists on your device and in your personal iCloud account.

Is this app affiliated with the IB?

No. This is an independent tool built by a teacher. It is not produced, endorsed, or affiliated with the International Baccalaureate Organization. Teachers should always refer to the official IB Design Technology Guide as the authoritative source.

I found a bug / something isn't working

Please email me with a description of the issue, which tab you were on, and what you expected to happen. Screenshots are very helpful. I'll investigate and release a fix as quickly as possible.

Can I request a new feature?

Absolutely. Send an email with your suggestion. This app is actively developed and teacher feedback directly shapes what gets built next.

Tab Guide

  • Roster — Student details, photos, SEN records, IB numbers
  • Grades — Paper 1, Paper 2, and IA scores with IB 1-7 grade prediction
  • Assessments — Custom tests and quizzes with exam prediction engine
  • IA — Per-criterion IA marking with student folder browser
  • IA Comments — Moderator-facing commentary for each IA criterion
  • Topics — Topic rubrics with 4-band performance descriptors
  • Reports — Comment bank with smart name/gender/level replacement
  • Letter — Full report letter generator with templates
  • EE — Extended Essay tracker for DT EE students
  • Curriculum — 2027 syllabus checklist with teaching progress
  • Guide — IA writing guide, study notes, and question bank

System Requirements

  • iPad app: iPad running iOS 16.0 or later
  • Desktop version: Chrome, Safari, or Edge (latest version recommended)
  • iCloud sync: Requires an iCloud account with iCloud Drive enabled
  • Storage: Minimal — the app is under 2 MB. Student data typically uses 50-500 KB.
